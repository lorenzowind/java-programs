O programa consiste em três tarefas principais:

1. Gerar relatório semanal indicando a quantidade de carros de cada categoria e as respectivas placas;

2. Calcular a quantidade de carros de alguma categoria especial que mais utilizou o estacionamento;

3. Descobrir se um ou mais clientes idosos retornaram ao estacionamento;

O projeto é estruturado da seguinte forma:

* Camada de apresentação (View): a classe MenuPrograma é responsável pela tratativa e validação
dos dados informados, sendo o ponto de partida do programa;

* Camada de controle (Controller): a classe ControladorTarefa é responsável pela manipulação dos
dados informados, mediando a entrada e saída das informações;

* Camada de modelo (Model): a classe ControleEstacionamento é responsável pelas operações lógicas,
gerenciando o comportamento e estrutura dos dados. Além disso, a classe Carro é um modelo que 
descreve os objetos utilizados dentro do contexto operacional do programa;

* Repósitórios: a interface RepositorioOperacoes é responsável pela definição e especificação 
dos métodos que devem ser implementados na camada de controle e modelo;

O trabalho contribuiu para compreensão dos seguintes conceitos:

* Implementação de variáveis compostas: utilização de arrays e matrizes dinâmicas, além de abordagens
de criação e manipulação para diferentes tipos de dados;
